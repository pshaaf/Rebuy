# Rebuy Poker App  

### A SwiftUI-Powered Poker Game Tracker  

Rebuy Poker App is a beautifully designed, intuitive, and feature-rich application that makes tracking poker game buy-ins, chip counts, and payouts seamless. Built with **SwiftUI**, it provides an efficient and visually appealing way to manage poker night finances while integrating with **Venmo** for quick transactions.

---

## Features  

**Player Management** ‚Äì Add, edit, and remove players dynamically. Each player has a name, buy-in amount, chip count, and Venmo status.  

**Real-Time Profit/Loss Calculation** ‚Äì Instantly see each player's winnings or losses with a color-coded display.  

**Game Logs & History** ‚Äì Keep a detailed record of past poker sessions, including buy-ins, chip totals, and financial outcomes.  

**Game-Saving** ‚Äì Game logs are automatically saved and persist between app sessions.  

**Smooth UI & Navigation** ‚Äì Modern SwiftUI components ensure a clean, user-friendly experience with **scrollable tables**, **custom buttons**, and **tap gestures for dismissing keyboards**.  

**Dark Mode Support** ‚Äì The app seamlessly adapts to both **light and dark mode** for a great visual experience.
